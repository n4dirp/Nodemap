# Nodes Minimap

Minimap overlay for the Blender Node Editor.

## Features

- Click, drag, scroll, and pan to navigate
- Hover and click to select nodes
- Zoom and pan inside the minimap
- Theme-aware colors

## Requirements

- Blender 5.1.0+

## Build from Source

```bash
git clone https://github.com/n4dirp/NodesMinimap.git
cd "Nodes Minimap/nodes_minimap"
blender --command extension build
```
